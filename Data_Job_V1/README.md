### ğŸ“Œ Project Overview
This project focuses on building an end-to-end **Power BI Dashboard** to analyze the global data job market. 

Rather than just looking at rows of data, I wanted to build a tool that helps understand the real value of different data roles. I used **Power Query** to clean and standardize raw job posting data, and then built an interactive dashboard that offers both a high-level market summary and a granular "deep dive" into specific job titles.

5. **Platform Efficiency:** Which platforms (LinkedIn, Indeed, etc.) actually have the most jobs?
